A UI test automation framework built for DemoWebShop using 
Selenium WebDriver, TestNG, and the Page Object Model (POM) design pattern. 
Supports data-driven testing via TestNG's @DataProvider.


***Tech Stack***
Tool  Version

Java     17
Selenium  WebDriver4.18.1
TestNG   7.11.0
ChromeDriver 147
Maven   Latest


***** Features *******
Page Object Model (POM) — clean separation of test logic and page interactions
TestNG Listeners — automatic pass/fail/skip logging for every test
Data Driven Testing — multiple registration scenarios via @DataProvider
BaseTest — centralized WebDriver lifecycle management (@BeforeMethod / @AfterMethod)
Implicit Waits — handles page load timing automatically.
